# ETH Samba Frontend Workshop (28/03/2023)

Setup the project:

```
yarn create vite ethsamba-frontend-workshop  --template react-ts
```

Install the dependencies:

```
yarn add @web3modal/ethereum @web3modal/react wagmi ethers@^5
```

Start the project

```
yarn dev
```
